# 🚀 Next.js Developer Portfolio

A modern, customizable developer portfolio built with **Next.js**, designed for tech professionals who want to showcase their projects, skills, and experience — with seamless integration to GitHub repositories via the official GitHub API.

---

## ✨ Features

- 📁 **GitHub Integration**  
  Automatically fetch and display your repositories using the GitHub API

- 🎨 **Modern UI/UX**  
  Clean and responsive design focused on readability and aesthetics

- ⚡ **Fast Performance**  
  Built with Next.js for optimized loading and SEO

- 🧩 **Modular Structure**  
  Easy to customize and extend

- 🌐 **Responsive Design**  
  Works across desktop, tablet, and mobile devices

---

## 🛠️ Tech Stack

This project is built using:

- **Next.js**
- **React**
- **TypeScript**
- **Tailwind CSS**
- **ShadCN UI**
- **GitHub REST API**

> You can easily extend it with other libraries as needed.

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/PedroMarqus/portif-lio-nextjs.git
```

Navigate into the project directory:

```bash
cd portif-lio-nextjs
```

Install dependencies:

```bash
npm install
# or
yarn install
```

---

## ▶️ Running the Project

Start the development server:

```bash
npm run dev
```

Then open:

```
http://localhost:3000
```

---

## 🔗 GitHub Integration Setup

To enable GitHub repository fetching:

1. Create a `.env` file in the root of the project
2. Add the following:

```env
GITAME=your_username
ROLE='your role'
```
